# House Price Prediction with Random Forest
Tujuan proyek ini adalah untuk memprediksi harga rumah secara akurat berdasarkan data historis dan fitur-fitur penting, guna mendukung pengambilan keputusan dalam industri properti.

Dataset Kaggle House Prices dapat diunduh [disini](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data)

Random Forest digunakan karena mampu menghasilkan prediksi yang akurat dan robust dengan menggabungkan banyak decision tree, sehingga mengurangi risiko overfitting, menangani data dengan fitur kompleks atau non-linear, dan memberikan hasil yang stabil meskipun terdapat variasi dalam data.
